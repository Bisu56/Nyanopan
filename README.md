# Nyanopan

Nyanopan is a personal website built for my dream to start an ecommerce business with my friend. This project represents our vision to create an online shopping platform.

## About the Project

Nyanopan is a full-featured ecommerce website built with Laravel. It includes everything you need to run an online store - from product management to checkout and order processing.

## Features

- **User Management**: User registration, login, and profile management
- **Product Catalog**: Browse products by category
- **Shopping Cart**: Add products to cart, update quantities, remove items
- **Checkout System**: Complete order checkout process
- **Order Management**: View and track orders
- **Admin Panel**: Manage products, categories, and orders
- **Contact System**: Customer complaints and contact form

## Tech Stack

- **Backend**: Laravel 9 (PHP 8.0+)
- **Frontend**: Blade Templates
- **Database**: MySQL
- **Authentication**: Laravel UI + Sanctum

## Installation

1. Clone the repository
2. Install dependencies:
   ```
   composer install
   ```
3. Copy `.env.example` to `.env` and configure database
4. Run migrations:
   ```
   php artisan migrate
   ```
5. Start the server:
   ```
   php artisan serve
   ```
